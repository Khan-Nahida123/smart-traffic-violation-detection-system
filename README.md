# Smart Traffic Violation Detection System

An end-to-end **Automatic Number Plate Recognition (ANPR)** system that detects vehicle license plates from images, extracts plate text using OCR, applies traffic violation rules, logs events into a database, and automatically sends fine notification emails.

The system integrates **Generative AI (LLM via Gemini API)** to dynamically generate human-readable fine notification messages instead of static templates.

This project demonstrates a **production-style Machine Learning + Backend + Automation pipeline** similar to real-world smart traffic enforcement systems.

# Problem Statement

Manual traffic monitoring is slow, error-prone, and difficult to scale.

This project simulates an automated system that:

- Detects vehicle license plates
- Reads plate text using OCR
- Applies traffic violation rules
- Logs violation records into a database
- Generates violation notifications using **Generative AI (LLM)**
- Sends automated fine notification emails

The goal is to build a **complete ML system, not just a standalone model**.

# System Architecture

![System Architecture](images/system_architecture.png)

```
Image Upload
      ↓
YOLOv8 Plate Detection
      ↓
OCR Text Extraction
      ↓
Violation Decision Engine
      ↓
LLM Fine Message Generation (Gemini API)
      ↓
Database Logging (MySQL)
      ↓
Automatic Email Notification
```

Each module is designed to be **modular and independently scalable**.

# Setup Instructions

### 1. Clone repository

```
git clone <your_repo_link>
cd anpr-project
```

### 2. Create virtual environment

```
python -m venv .venv
.venv\Scripts\activate
```

### 3. Install dependencies

```
pip install -r requirements.txt
```

### 4. Configure environment

Create `.env` file:

```
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=anpr_db

SMTP_USER=your_email@gmail.com
SMTP_PASS=your_app_password

GEMINI_API_KEY=your_api_key
```

---
